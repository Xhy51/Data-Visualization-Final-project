Green Impact SF

by Martin Xu

Project Description

Green Impact SF is an interactive dashboard designed to analyze the impact of urban green spaces on environmental quality in San Francisco. The project utilizes various data visualization techniques, including an interactive choropleth map, time-series analysis charts, scatter plot matrices, and heat maps to provide comprehensive insights. Data processing is performed in Python while the visualizations are rendered using D3.js.

Interactive Visualization

The visualization area will display dynamic D3.js charts and maps.
